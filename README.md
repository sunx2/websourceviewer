# websourceviewer
A webapp written in Django and Python to scrape contents of a website and display it


A simple project created by me while learning Django and deploy with heroku

**requirements**
- [x] Python 3.5+ 
- [x] BS4 module
- [x] Requests module
- [ ] lxml parser
- [x] Django
- [x] Dj-database-url==0.3.0
- [x] dj-static==0.0.6
- [x] gunicorn==19.6.0
- [x] Unipath==1.0
- [x] python-decouple==3
- [x] Pillow==3.3.0
- [ ] whitenoise==3.2
- [x] psycopg2==2.6.1
